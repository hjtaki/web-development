CSS Positioning: Relative

Use this tag to demonstrate position:relative;

Relative positioning moves a box model to a new location relative to where it would appear in normal (static) flow.

One or more coordinates may be applied to move a box model (top, bottom, left, right)

  • begin with the css rule. eg: position:relative;
  • apply one or two coordinates (top, bottom, left, right) to describe how to shift the location of the box. eg: top:40px;
  • use only with block elements
  • often applied with classes and ids
  • other elements on the page are still affected by its presence (relative positioned elements remain a part of content flow)
