CSS Positioning: Combining Absolute and Relative

Box B

There are countless ways to combine both relative and absolute positioning. Here's one useful technique: Since absolute positioned elements are placed based on the edges of the space made available by the first non-static ancestor, if absolute positioned box A is contained in a relative (non-static) positioned box B, then A is absolute positioned based on Bs edges.

If an absolute positioned tag has no non-static ancestor, it will use the edges of the browser view port as the boundary for determining which corner to 'stick' to.

  • assign position:relative; to the parent container
  • assign position:absolute; to the child element
  • apply to the child one or two coordinates (top, bottom, left, right) to describe which corner of the parent to 'stick' to. eg: top:40px; left:10px;
