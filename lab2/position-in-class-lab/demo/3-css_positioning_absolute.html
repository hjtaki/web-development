CSS Positioning: Absolute

Use this tag to demonstrate position:absolute;

Absolute positioning places the box model according to the edges of the available space. The available space is determined by the first parent element with a position other than static. If no such element exists then the available space is determined by the containing <html> tag.

When absolute positioning, at least two coordinates are usually described, an X axis (right or left) and a Y axis (top or bottom).

  • begin with the css rule position:absolute
  • apply one or two coordinates (top, bottom, left, right) to describe how to shift the location of the box. eg: top:40px; left:10px;
  • use only with block elements
  • often used with divs
  • applied using ids (unique on the page)
  • removes the element from 'content flow', eg: (it no longer affects other element positions)
  • will 'stick' to the corner of the first non-static ancestor it finds, or use the browser edge if no non-static ancestor exists.
    • once positioned, absolute elements will follow the page scroll
